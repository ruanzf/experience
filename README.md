二、优化问题

	1、优化业务
		业务上一般都会有好多解决方案，例如批量、线程池、链接池、去掉没必要的操作、把问题从上游过滤掉等
	2、优化系统
		调整JVM、mysql等系统参数、调整索引、优化耗时操作、优化java代码促成内联、即时编译等
	3、优化架构
		针对业务使用中间件、业务拆分、分表分库、分布式等
	4、优化算法
		以空间换时间、调整数据结构等。
	5、加资源
		升级硬件
	
	一个系统往往会受多维度影响，所谓鱼和熊掌不可得兼，但是可以根据自己的业务找到最佳平衡点
	
三、科学家创造工具。开发者能深入了解工具，对比工具并根据业务选出合适的工具。
